HOW TO UPDATE GITHUB
1. Open your GitHub repository: kingdom-frequency-radio-app
2. Delete the old app files or upload these new files and replace matching names.
3. Make sure every file from this ZIP is in the repository root.
4. Commit the changes.
5. GitHub Pages will redeploy automatically.
6. Open https://app.kingdomfrequencyradio.com and refresh.
