KINGDOM FREQUENCY RADIO — FREE WEB APP

WHAT IS INCLUDED
- Live Zeno.fm radio player
- Installable home-screen app
- iPhone installation instructions
- Share button
- Offline app shell (the live radio itself still requires internet)
- App icons

FREE PUBLISHING WITH GITHUB PAGES
1. Create a free GitHub account at github.com.
2. Create a new PUBLIC repository named: kingdom-frequency-radio-app
3. Upload every file from this folder to the repository.
4. Open repository Settings > Pages.
5. Under "Build and deployment," choose "Deploy from a branch."
6. Select the main branch and /(root), then save.
7. GitHub will provide an HTTPS address for the app.

IPHONE INSTALLATION
1. Open the published app link in Safari.
2. Tap the Share icon.
3. Tap "Add to Home Screen."
4. Tap Add.
